# Jogo da Velha

Este é um simples jogo da velha desenvolvido em JavaScript. Ele permite que dois jogadores joguem o clássico jogo da velha em um navegador da web.

## Funcionalidades

- Permite que dois jogadores joguem alternadamente.
- Verifica automaticamente quando um jogador vence ou quando o jogo termina em empate.
- Reinicia o jogo após a conclusão de uma partida.

## Como Jogar

1. Clone este repositório em sua máquina local: https://github.com/vitorpaiv4/Jogo-da-Velha
   
2. Abra o arquivo `index.html` em seu navegador da web.

3. O jogo da velha será exibido na tela. Clique em uma célula vazia para fazer sua jogada.

4. Os jogadores alternam as jogadas, marcando células vazias com "X" ou "O".

5. O jogo continuará até que um jogador vença ou haja um empate. Nesse momento, uma mensagem será exibida na tela indicando o resultado.

6. Para iniciar um novo jogo, clique no botão "Reiniciar".

## Estrutura do Projeto

- `index.html`: Contém a estrutura HTML do jogo da velha, incluindo o tabuleiro e os botões de reiniciar.
- `style.css`: Arquivo CSS para estilização do jogo.
- `script.js`: Arquivo JavaScript para manipulação da lógica do jogo.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
